# 3D gul modellari (GLB)

Shu papkaga GLB tashlang — UI avtomatik ishlatadi, hech qanday kod o'zgarmaydi.
Fayl bo'lmasa premium protsedural gul ko'rsatiladi (hozirgi holat).

Qidiriladigan nomlar (tartib bo'yicha birinchisi olinadi):

- `peony.glb`
- `rose.glb`
- `tulip.glb`
- `lily.glb`
- `flower.glb`

GLB ichida animatsiya klipi bo'lsa — dvigatel avtomatik topib, yumshoq crossfade
bilan ijro etadi (har nusxa ozgina boshqa tezlikda). Klip bo'lmasa protsedural
shamol/nafas animatsiyasi qo'llanadi.

Tavsiyalar:
- Draco yoki meshopt bilan siqilgan, ≤ 2–3 MB
- PBR materiallar (roughness/metalness), 1K–2K teksturalar
- Gul markazi koordinata boshida, poya pastga (−Y) qarab
- Balandligi ~2 birlik atrofida (sahna masshtabiga mos)

Manba g'oyalari: Sketchfab (CC litsenziyali "peony/rose photoscan"), Polyhaven.
