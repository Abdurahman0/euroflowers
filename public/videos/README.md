# Kinematik fon videolari

Shu papkaga video tashlang — login sahifasi avtomatik ishlatadi (past opacity + blur,
chalg'itmaydi). Fayl bo'lmasa video qatlami umuman ko'rinmaydi — hech narsa buzilmaydi.

Qidiriladigan nomlar:

- `garden.webm` (afzal — kichik hajm)
- `garden.mp4`

Tavsiyalar:
- Slow-motion gul bog'i / shamolda tebranayotgan gullar / tushayotgan gulbarglar
- 1080p yetarli, ≤ 8–10 MB, ovozsiz
- Juda sekin harakat — "stock video" emas, atmosfera
- Manba: Pexels/Pixabay "flower garden slow motion" (bepul litsenziya)
